This folder contains the code to create a React application that is a clone of Slingshot with the implemented
Slinger feature. The components folder contains the code for the individual components that make up the 
task and project pages. The css folder contains the css files that style the page components, the images
folder contains the images used in the pages, the mainfiles page has the container for the entire application
and loads the top navigation bar, the left navigation bar, and the workspace which changes based on 
whether the user clicks the projects or tasks button. The pages folder contains the individual workspaces: the
tasks page and the projects page
